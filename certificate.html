Open_ssl

  • To generate a public key from a private key openssl rsa -in private.key -pubout -out public.key
  • To check a certificate openssl x509 -in certificate.crt -text -noout
  • To check an SSL connection openssl s_client -connect host:port
  • To check a mTLS connection openssl s_client -connect host:port -cert client.crt -key client.key -CAfile ca_cert.pem
  • To get cacerts of a server openssl s_client -showcerts -connect host:port
  • Check if private key and cert are related
    • first with the certificateopenssl x509 -noout -modulus -in tls.crt | openssl md5
    • then with the keyopenssl rsa -noout -modulus -in tls.key | openssl md5

ssh-keygen

  • To generate a pair of ssh keys ssh-keygen -t rsa -b 2048 -f my_ssh_key

curl

  • Very verbose simple curl curl -vvv https://host
  • Check mTLS curl -X POST/GET -H "Content-Type: application/json" https://host/servicePath --cacert cacert.pem --cert cert.pem --key key.key

nmap

  • To list all the ciphersuites that a server accepts nmap --script ssl-enum-ciphers -p port host
