Python Scripts

- for secrets

  • To extract any secrets values from a cluster in base64. Run python getAllSecretsInBase64.py namespace

- for keystore and trustore

  • To create keystore with ibm (stet) key must be equal to pwd of the private key. Type python cert_to_keystore.py, then answer the question. Alias is ibm
  • To create trustore. Type phyton cert_to_truststore.py then answer the question, alias can be set up. Run it several time to add several entries
  • To create keystore with 2 alias (kafka and file). Open the script to change ns. Type python cert_to_keystore_multi.py.
