Open_ssl

  • To generate a public key from a private key openssl rsa -in private.key -pubout -out public.key
  • To check a certificate openssl x509 -in certificate.crt -text -noout
  • To check an SSL connection openssl s_client -connect www.example.com:443
  • To check a mTLS connection openssl s_client -connect host:port -cert client.crt -key client.key -CAfile ca_cert.pem

ssh-keygen

  • To generate a pair of ssh keys ssh-keygen -t rsa -b 2048 -f my_ssh_key
