MongoDB

  • To use a database use database_name
  • To show collections show collections
  • To group by and count db.collection_name.aggregate([ { $group: { _id: "$city", total: { $sum: 1 } } } ]);
  • To display by order ascending db.collection_name.find().sort({ age: 1 });
  • To display by order descending db.collection_name.find().sort({ age: -1 });
